### 快速开始

#### 官方教程链接

[开始使用_飞桨 - 源于产业实践的开源深度学习平台](https://www.paddlepaddle.org.cn/install/quick?docurl=/documentation/docs/zh/develop/install/pip/linux-pip.html)

#### 1. 虚拟环境官方支持

官方支持的 Python 版本：3.9 / 3.10 / 3.11 / 3.12 / 3.13

#### 2. 完成 PaddlePaddle 的安装

根据所选 CUDA 配置版本安装，以下是不同 CUDA 版本的安装命令：

- 

  **CUDA 11.8 的 PaddlePaddle**

  ```
  python3 -m pip install paddlepaddle-gpu==3.2.2 -i https://www.paddlepaddle.org.cn/packages/stable/cu118/
  ```

- 

  **CUDA 12.6 的 PaddlePaddle**

  ```
  python3 -m pip install paddlepaddle-gpu==3.2.2 -i https://www.paddlepaddle.org.cn/packages/stable/cu126/
  ```

#### 3. 安装 paddleocr

```
python -m pip install paddleocr
```

安装后会有一个 `.paddlex`文件夹，示例结构如下：

```
.paddlex
├── fun_cet
├── locks
└── official_models
    ├── PP-LCNet_x1_0_doc_ori
    ├── PP-LCNet_x1_0_textline_ori
    ├── PP-OCRv5_server_det
    ├── PP-OCRv5_server_rec
```

需将其中的 `PP-OCRv5_server_rec`替换为仓库里上传的模型。

#### 4. 改动相关代码路径

根据实际路径修改以下代码：

- 

  **`evaluate_train_rec.py`**

  ```
  # 数据集目录
  DATASET_ROOT = Path("/home/zwm/datasets")
  
  ALL_TEST = DATASET_ROOT / "splits/all_test.txt"
  ALL_HARD = DATASET_ROOT / "splits/all_hardtest.txt"
  
  # 识别推理模型目录（inference.json / inference.pdiparams / inference.yml）
  REC_MODEL_DIR = Path("~/.paddlex/official_models/PP-OCRv5_server_rec_old").expanduser()
  ```

- 

  **`evaluate_train.py`**

  ```
  DATASET_ROOT = Path("/home/zwm/datasets")
  
  ALL_TEST = DATASET_ROOT / "splits/all_test.txt"
  ALL_HARD = DATASET_ROOT / "splits/all_hardtest.txt"
  
  DET_DIR = Path("~/.paddlex/official_models/PP-OCRv5_server_det").expanduser()
  REC_DIR = Path("~/.paddlex/official_models/PP-OCRv5_server_rec").expanduser()
  ```

#### 5. 运行脚本

```
python evaluate_train.py
python evaluate_train_rec.py  # 仅识别
```
